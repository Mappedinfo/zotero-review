# Zotero Review - 文献评审管理功能使用指南

## 功能概述

Zotero Review 插件为 Zotero 7 添加了强大的文献评审管理功能，特别适用于系统文献综述（Systematic Literature Review）等需要对大量文献进行筛选和评审的场景。

## 核心功能

### 1. 专用评审表格视图

- **独立的Tab页面**：在 Zotero 主窗口中创建专门的文献评审表格视图
- **完整的元数据展示**：显示文献的ID、标题、作者、年份、期刊等核心信息
- **自定义评审字段**：支持添加任意数量的自定义字段来记录评审信息

### 2. 灵活的字段管理

- **动态字段配置**：可随时添加、编辑或删除评审字段
- **多种字段类型**：
  - 文本：用于记录评审备注、关键词等
  - 下拉选项：用于评级、分类等标准化评审
  - 数字：用于评分
  - 日期：用于记录评审时间
  - 是/否：用于快速标记是否纳入等决策

### 3. 实时数据编辑

- **表格内直接编辑**：在表格中直接修改评审数据，自动保存
- **数据持久化**：所有评审数据存储在 Zotero 首选项中，不会丢失
- **与文献关联**：评审数据与文献ID绑定，即使文献信息更新也不影响

### 4. 强大的导出功能

- **CSV格式**：兼容 Excel、Google Sheets 等常用工具
- **JSON格式**：便于数据分析和程序处理
- **包含完整信息**：导出的数据包含文献元数据和所有评审字段

## 快速开始

### 安装插件

1. 下载最新版本的 `.xpi` 文件
2. 在 Zotero 中，选择 `工具` > `插件`
3. 点击右上角的齿轮图标，选择 `Install Add-on From File...`
4. 选择下载的 `.xpi` 文件并安装
5. 重启 Zotero

### 打开评审视图

有两种方式打开文献评审视图：

1. **通过菜单**：`工具` > `打开文献评审视图`
2. **通过快捷键**：（可在首选项中配置）

### 配置评审字段

首次使用时，插件已预设了一些常用字段：

- **相关性**：高/中/低/不相关
- **质量评分**：A/B/C/D
- **是否纳入**：是/否
- **评审备注**：文本

您可以根据需要自定义这些字段：

1. 在评审表格视图中，点击工具栏的 `管理字段` 按钮
2. 在弹出的对话框中：
   - 点击 `添加新字段` 添加字段
   - 修改字段名称、类型和选项
   - 点击 `删除此字段` 移除不需要的字段
3. 点击 `保存` 应用更改

### 进行文献评审

1. 在评审表格中，每一行代表一篇文献
2. 点击某一行可以在主窗口中定位并查看该文献的详细信息
3. 在自定义字段列中：
   - **下拉选项**：点击下拉框选择相应的值
   - **文本/数字**：直接输入内容
   - **是/否**：勾选或取消勾选
4. 所有修改会自动保存

### 导出评审数据

1. 点击工具栏的 `导出` 按钮
2. 选择导出格式：
   - **导出为 CSV**：用于在 Excel 中进一步分析
   - **导出为 JSON**：用于编程处理或数据分析
3. 选择保存位置
4. 完成导出

## 使用场景

### 系统文献综述

1. **初筛阶段**：
   - 添加"初筛结果"字段（通过/不通过）
   - 添加"排除原因"字段（文本）
   - 快速标记每篇文献的初筛结果

2. **全文评审阶段**：
   - 添加"质量评估"字段（高/中/低）
   - 添加"方法学质量"字段（A/B/C/D）
   - 添加"数据提取"字段（文本）
   - 详细记录评审信息

3. **最终纳入阶段**：
   - 使用"是否纳入"字段标记最终决策
   - 导出最终纳入的文献列表
   - 生成评审统计报告

### 课题文献管理

1. 添加"阅读状态"字段（未读/已读/重点）
2. 添加"相关度"字段（高度相关/相关/参考）
3. 添加"研究方法"字段（实验/问卷/理论等）
4. 添加"笔记"字段记录重要信息

### 学位论文文献整理

1. 添加"使用章节"字段（引言/文献综述/讨论等）
2. 添加"重要性"字段（1-5星级）
3. 添加"引用方式"字段（直接引用/间接引用/背景）

## 高级功能

### 批量操作

虽然目前版本不支持批量编辑，但您可以：

1. 导出数据为 CSV
2. 在 Excel 中批量编辑
3. 使用导入功能（未来版本支持）

### 数据备份

评审数据存储在 Zotero 的首选项中，建议定期：

1. 导出评审数据为 JSON 格式备份
2. 使用 Zotero 的同步功能备份整个数据库

### 多人协作

目前版本的评审数据是本地存储的，如需多人协作：

1. 定期导出评审数据
2. 使用版本控制系统（如Git）管理导出的数据
3. 或使用共享文件夹同步导出的数据

## 常见问题

### Q: 评审数据存储在哪里？
A: 评审数据存储在 Zotero 的首选项（preferences）中，与 Zotero 数据库分离，不会影响原始文献信息。

### Q: 卸载插件后数据会丢失吗？
A: 是的，卸载插件前请务必导出评审数据。

### Q: 可以在多台电脑上同步评审数据吗？
A: 目前评审数据不会自动同步。建议导出数据并手动同步，或等待未来版本的云同步功能。

### Q: 表格显示很慢怎么办？
A: 如果文献数量非常大（超过10000篇），建议使用 Zotero 的集合功能先筛选出需要评审的文献子集。

### Q: 如何修改已有字段的类型？
A: 在列管理器中可以修改字段类型，但已有数据可能需要重新输入。

### Q: 支持搜索和筛选功能吗？
A: 当前版本表格支持基本的显示功能，高级搜索和筛选功能将在后续版本中添加。

## 技术规格

- **兼容性**：Zotero 7.0+
- **数据格式**：JSON (内部存储) / CSV / JSON (导出)
- **字段类型**：text, select, number, date, boolean
- **存储位置**：Zotero Preferences
- **最大字段数**：无限制（建议不超过20个以保持性能）
